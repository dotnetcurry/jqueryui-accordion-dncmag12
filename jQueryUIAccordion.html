What Does This Book Contain?

The Absolutely Awesome jQuery CookBook contains scores of practical jQuery recipes that you can use in your projects right away, this cookbook helps you gain hands-on experience with the jQuery API. Each recipe includes working code, a live demo and a discussion on why and how the solution works. This eBook covers the latest version of jQuery 1.11.0, jQuery UI 1.10.4 & jQuery 2.0.
